Frequently Asked Questions about the VizCoSH module

  • Why only two category-levels?
    Two levels are generally enough for all textbooks, three levels would lead to poorly structured documents. The VizCoSH module is designed for creation of short multipage study materials.

  • Can students edit VizCoSHs?
    Only teachers can create and edit VizCoSHs. There are no plans to implement student editing for textbooks. The main reason is to keep the VizCoSH-module as simple as possible.

  • How do I search the chapters?
    At present there is only one way, use browser's search capability in print page. Global searching is now possible only in Moodle forums.

  • My titles do not fit on one line.
    Either rephrase your titles or ask your site admin to change TOC width. It is defined globally for all vizcoshs in module configuration page.

  • What exactly is eMargo?
    eMargo supports interactive communication with other participants of a lecture or with its supervisors. When using this feature, users can annotate existing lecture notes in an extra column next to the main text and are able to add comments.
    Teachers are able to disable the eMargo-features in every VizCoSH if the annotation features are not needed.

  • How can I change the size of the comment-column?
    It is possible to change the height and width of the comment-column on the right side in your global Moodle-administration-area. If you don't have access to it, ask your administrator.
