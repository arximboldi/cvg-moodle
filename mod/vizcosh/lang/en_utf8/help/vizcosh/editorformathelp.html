Algorithm-Visualization-Format Editor

Here you can create your own algorithm visualization format. It will be available for everyone.

You need to specify a name and optionally a file extension which should be the extension of the files containing algorithm visualization data. Most importantly you need to give a JNLP file template which will be used to create the actual jnlp file which then will be started by the user. A JNLP Template can have variables which will be replaced by the appropriate values:

  • JNLP-PATH: the path to the jnlp-file
  • JNLP-FILENAME: the filename of the jnlp-file
  • DATA-PATHFILENAME: the filename and path to the file containing algorithm visualization data
  • DATA-TYPE: the type of the file containing algorithm visualization data
  • JAR-PATH: the path to the jar-files which will be used by this jnlp

There are three JNLP-templates given which can also be used or edited.
