Annotationsmöglichkeiten mit eMargo

Das VicCoSH-Modul unterstützt die Einbindung von Annotationsfunktionen über das so genannte "eMargo"-System, das eine digitale Randspalte für die Annotation von Texten anbietet. Eine detaillierte Beschreibung aller Annotationsfunktionen des eMargo-Systems kann über den Hilfe-Button direkt in der eMargo-Randspalte geöffnet werden.
