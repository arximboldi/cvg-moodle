Algorithmus-Visualisierungs-Editor

Hier kann eine eigene Algorithmus-Visualisierung erstellt werden. Diese wird dann für alle verfügbar sein.

Es müssen ein Name und eine Beschreibung, sowie einige passende Schlagwörter (für die Suche, optional) angegeben werden. Außerdem muss ein Format angegeben werden, das angibt, wie die Visualisierung gestartet werden soll. Weiterhin kann eine Datei mit für die Algorithmus-Visualisierung relevanten Daten hochgeladen werden. Optional kann ein Bild angegeben werden, das angezeigt wird, wenn die Visualisierung zu einem VizCoSH-Kapitel hinzugefügt wird.
